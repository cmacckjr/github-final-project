# github-final-project
This repo is being created for my final project in my "Getting Started with Git and GitHub" class contained in the IBM Full-Stack JavaScript Developer Professional Certificate course offered by IBM through Coursera.

A calculator that calculates simple interest given principal, annual rate of interest and time period in years.
Input:
    p, principal amount
    t, time period in years
    r, annual rate of interest
Output:
    simple interest  = p*t*r
